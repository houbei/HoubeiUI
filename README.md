# HoubeiUI框架

* 后备网HoubeiUI框架是用HTML、CSS 、JAVASCRIPT和部分开源插件开发的一个响应式前段框架（前台、后台）,HoubeiUI框架满足开发中的项目需求,可以将需要完成的任务代码量最小化，大大提高项目开发效率与质量。

* 做为优秀的框架产品,在系统性能上做的大量的优化处理,只为让程序员使用HoubeiUI框架强悍的功能同时,用最短的时间完成项目的开发。
